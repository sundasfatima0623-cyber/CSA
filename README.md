# Compassionate St. Augustine — Website

A multi-page website for **Compassionate St. Augustine (CSA)**, a 501(c)(3) nonprofit influencing, inspiring, and growing a culture of compassion in St. Augustine, Florida through advocacy, awareness, and action.

## Pages

- `index.html` — Home / Landing
- `gift-gardens.html` — Gift Gardens program
- `compassionate-women.html` — Compassionate Women program
- `good-trouble.html` — Good Trouble advocacy
- `purple-bench.html` — The Purple Bench business recognition
- `casa.html` — CASA: Compassionate Aging St. Augustine (new initiative)
- `events.html` — Events calendar

## Shared Assets

- `style.css` — Brand styling and component library
- `script.js` — Mobile nav + scroll fade-in animations
- `assets/logo.jpeg` — Bridge of Lions logo

## Brand System

- **Primary background:** `#1E3A45` (deep teal-navy)
- **Section alternate:** `#162E38`
- **Accent warm:** `#C4875A` (terracotta)
- **Accent light:** `#A8C5C0` (seafoam)
- **Text on dark:** `#F0EDE8`
- **Headings:** Cormorant Garamond
- **Body:** Source Sans 3 (minimum 18px for accessibility)

## Local Preview

This is a static site — no build step required. To preview locally, just open `index.html` directly, or run a quick static server:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.
